# Student Portable Download Instructions

Use this if your instructor provides BDD Studio portable release files.

## macOS Apple Silicon

For M1/M2/M3/M4 Macs, download:

```text
BDDStudio-macos-arm64.zip
```

Unzip it and double-click:

```text
BDD Studio.command
```

If macOS blocks it:

```text
Right-click BDD Studio.command
Choose Open
Confirm Open
```

## macOS Intel

For older Intel Macs, download:

```text
BDDStudio-macos-x64.zip
```

Unzip it and double-click:

```text
BDD Studio.command
```

## Windows

Download:

```text
BDDStudio-windows-x64.zip
```

Unzip it and double-click:

```text
BDD Studio.bat
```

If Windows SmartScreen appears:

```text
More info
Run anyway
```

## Linux

Download:

```text
BDDStudio-linux-x64.tar.gz
```

Extract it:

```bash
tar -xzf BDDStudio-linux-x64.tar.gz
cd BDDStudio-linux-x64
chmod +x run-bddstudio.sh
./run-bddstudio.sh
```

## Browser Address

BDD Studio opens at:

```text
http://127.0.0.1:8765
```

If that port is busy, close the old BDD Studio window or ask your instructor for help.

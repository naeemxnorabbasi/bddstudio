BDD Studio Portable Package

1. Unzip or extract this package.
2. Run the launcher for your operating system.

macOS:
  Double-click "BDD Studio.command"
  If macOS blocks it, right-click it and choose Open.

Windows:
  Double-click "BDD Studio.bat"

Linux:
  Run:
    chmod +x run-bddstudio.sh
    ./run-bddstudio.sh

BDD Studio opens in your browser at:

  http://127.0.0.1:8765

If the port is already busy, start BDD Studio manually with a different port:

  bddstudio serve --port 8766

Files stay local on your computer.
